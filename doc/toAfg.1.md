# NAME

toAfg - Print reads stored in a file (pls|fasta|fastq) as an afg.

# SYNOPSIS
**toAfg** *input.filetype* *output.filetype*
	[**-minSubreadLength** *l*]  [**-regionTable** *regions_file*]
	[**-noSplitSubreads**]  [**-useccsdenovo**]  [**-uniformQV** *QV*]
